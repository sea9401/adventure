# 아이템 도감

게임 내 모든 아이템을 카테고리별로 정리한 표입니다. 새 아이템을 코드에 추가할 때마다 이 문서도 함께 갱신합니다.

> **제작 품질 등급**: 대장간에서 제작한 장비에는 품질 등급(불량 / 하급 / 일반 / 고급 / 걸작)이 붙어, 주력 옵션이 등급에 따라 위아래로 흔들립니다(불량 −2 .. 걸작 +2). 아래 표의 효과는 **`일반` 등급**(= 분포의 평균) 기준이며, 변동이 있는 장비는 설명 끝에 `품질 변동: …` 로 적습니다.
>
> **드랍 품질 등급**: 몬스터·보스가 떨구는 **장비**는 드랍 시점에 품질이 **위로만** 굴려집니다 — 대부분 그대로(기본 ≈95%), 가끔 「**정교한** ○○」(주력 옵션 +1, ≈4%), 드물게 「**빼어난** ○○」(+2, ≈1%). 보스·고티어 몬스터는 좋은 품질 가중치가 올라갑니다(`dropQualityBias`). 아래 표의 효과는 **`기본` 등급**(= 최저 보장치) 기준입니다 — 제작산은 표시값=평균, 드랍산은 표시값=하한. 제작 등급과는 별개 명칭·별개 가방 칸이며, 퀘스트·상점·시작 장비는 변동 없이 표의 값 그대로입니다.
