# 아이템 도감

게임 내 모든 아이템을 카테고리별로 정리한 표입니다. 새 아이템을 코드에 추가할 때마다 이 문서도 함께 갱신합니다.

## 무기

| 이름 | 효과 | 설명 |
| --- | --- | --- |
| 나무 막대 | 공격력 +0 | 나뭇가지를 대충 다듬어 만든 평범한 막대. |
| 야구 방망이 | 공격력 +2 | 단단한 나무를 깎아 만든 묵직한 방망이. |
| 못박힌 야구방망이 | 공격력 +3, 활력 +1 | 방망이 끝에 낡은 못을 잔뜩 박아 넣었다. 휘두를 때마다 묵직하다. (제작서: 주정뱅이 0.3%) |
| 예비 손도끼 | 공격력 +2, 민첩 +1 | 나무꾼 지미가 챙겨 다니던 예비 손도끼. 손에 익으면 제법 매섭다. (지미 산적 의뢰 보상) |
| 산적의 단검 | 공격력 +4, 민첩 +2 | 산적이 품에 숨기고 다니던 단검. 짧지만 손에 착 감긴다. (산적 0.5%) |
| 두더지왕의 드릴 | 공격력 +5, 속도 +2 | 어느 두더지가 품에 꼭 쥐고 있던 작은 드릴. 회전시키면 묘하게 손맛이 좋다. (두더지 0.02%) |
| 수정 단검 | 공격력 +5, 민첩 +1 | 단단한 수정을 깎아 만든 날카로운 단검. (제작서: 동굴뱀 0.4%) |
| 골렘의 망치 | 공격력 +7, 속도 -2 | 부서진 골렘의 팔에서 떼어낸 둔중한 돌망치. 휘두르려면 두 손이 필요하다. (부서진 골렘 0.1%) |
| 마정석 검 | 공격력 +7, 힘 +3 | 마정석을 칼날에 박아 넣은 한손검. (제작서: 광맥의 수호자 25%, 4종 중 1) |
| 마정석 방패 | 공격력 +7, 활력 +3 | 마정석을 박은 묵직한 방패. 막으며 받아치는 데에도 쓴다. (제작서: 광맥의 수호자 25%, 4종 중 1) |
| 마정석 창 | 공격력 +7, 민첩 +3 | 끝에 마정석을 깎아 박은 긴 창. (제작서: 광맥의 수호자 25%, 4종 중 1) |
| 마정석 너클 | 공격력 +7, 행운 +3 | 마정석 조각을 손등에 박은 너클. (제작서: 광맥의 수호자 25%, 4종 중 1) |

## 방어구

| 이름 | 효과 | 설명 |
| --- | --- | --- |
| 천 옷 | 방어력 +0 | 평범한 천으로 만든 옷. |
| 물컹물컹한 갑옷 | 방어력 +3 | 슬라임 핵을 심으로 두른 갑옷. 충격을 부드럽게 흡수한다. |
| 박쥐가죽 후드 | 방어력 +1, 속도 +2 | 박쥐 가죽을 이어 만든 후드. 어둠 속에서도 발이 가볍다. (제작서: 박쥐 0.4%) |
| 비단 로브 | 방어력 +2, 행운 +4 | 거미줄을 비단처럼 곱게 짜낸 로브. 걸치고 있으면 묘하게 운이 따른다고 한다. (제작서: 거미 0.3%) |
| 망령의 망토 | 방어력 +2, 민첩 +1, 속도 +2 | 떠도는 망령이 두르고 있던 누더기 망토. 입으면 발걸음이 어딘가 가벼워진다. (떠도는 망령 0.2%) |
| 골렘갑주 | 방어력 +7, 속도 -3, 행운 -1 | 골렘의 잔해를 덧대어 만든 두꺼운 갑주. 묵직한 만큼 발걸음과 운이 따라 무거워진다. (제작서: 부서진 골렘 2%) |

## 장신구

| 이름 | 효과 | 설명 |
| --- | --- | --- |
| 엄마가 준 부적 | 행운 +2 | 어머니의 사랑이 깃든 작은 부적. |
| 활력의 반지 | 활력 +2 | 은은한 녹빛이 도는 반지. 끼고 있으면 몸이 가볍다. |
| 님프의 반지 | 속도 +2 | 은은하게 푸른빛이 도는 가는 반지. 호수 님프의 가호가 깃들어 있다. (호수 님프 0.5%) |
| 요정의 가호 | 활력 +3, 행운 +2 | 활력의 반지에 요정가루의 가호를 입힌 것. 끼고 있으면 몸도, 운도 따른다. (제작서: 호수 님프 0.2%) |
| 마정석 팔찌 | 활력 +3, 속도 +2 | 마정석 조각을 엮어 만든 팔찌. 손목에 두르면 몸이 단단해지면서도 발이 가벼워진다. (제작서: 광맥의 수호자 15%) |
