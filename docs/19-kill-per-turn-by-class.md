# 1킬/턴 스펙 가이드 — 직업별

> 탐험(파견) 중 **턴당 1마리 처치**를 유지하기 위한 기본 ATK 요구치.
> 모든 수치는 `src/lib/game/data.ts` / `src/lib/game/logic.ts` 기준. 변경 시 함께 업데이트.

## 1. 메커니즘 요약

- 매 턴: 적 1마리와 교전 → 처치 시 즉시 다음 적 등장 (`logic.ts:1154`)
- 적은 두 종류 중 **균등 랜덤** (`logic.ts:494`)
- 1턴에 1킬을 안정적으로 내려면 → **HP가 더 높은 쪽 적도 1턴에 잡을 만큼**의 데미지가 필요

### 데미지 공식 (`logic.ts:441`)

```
dmg = max(1, ATK × (1+atkBoost) − effectiveDEF)
    × (0.9 ~ 1.1)   // 균등 랜덤 변동, 평균 1.0
    × critMult       // 패시브/스킬 크리
    × multiplier
    × (1 + dmgAmp)
```

`effectiveDEF = enemy.def × (1 − defPierce)` — 마법사/원소술사는 `defPierce = 0.5`.

### 공격 횟수 (`logic.ts:390`)

```
attacks = 1 + floor(SPD × 0.05)   // cap 4
```

| SPD   | 공격/턴 |
| ----- | ------- |
| 0~19  | 1       |
| 20~39 | 2       |
| 40~59 | 3       |
| 60+   | 4       |

> 공격 횟수가 늘어나면 필요 ATK는 거의 비례해서 줄어든다 (`ATK ≥ HP/n + DEF`).

## 7. 메모

- **변동 ±10%**: 단발 1킬을 보장하려면 표 값에 **+10% 정도 마진**을 두는 게 안전 (변동이 0.9 나올 때도 처치하려면).
- **방패병**은 ATK 성장 ×0.55라 동렙 광전사 대비 ATK가 약 절반 수준 → 1킬/턴 유지가 가장 어려움. 대신 반사로 추가 데미지가 들어가지만 표에 반영 안 함 (변수 큼).
- **맹독술사**: 도트가 누적되어 시간차로 처치 — "1턴 1킬" 모델보다 "고정 DPT + DOT 누적" 모델로 봐야 함. 현 표에 직접 비교 불가.
- **원소술사**: 매 턴 INT×0.3 마력 분출(`magic_amp_with_aura` 패시브)이 추가 — 고INT일 때 적 MDEF 통과량만큼 누적. 표는 기본 공격만 반영.
- **다중 공격**과 **크리 멀티**는 곱연산이라 함께 쌓이면 효과가 폭발적임. 어쌔신 + SPD 60 = 표4 값의 **1/(1.75 × 4) ≈ 1/7** 수준.
- 설원/해적 이후부터는 **세트 효과 + 명예의 전당 + 룬**의 누적이 필수. 이 표는 "장비 0" 베이스라인 참고용.
