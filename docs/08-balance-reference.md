> 시뮬레이션 / 데미지 검증 / 밸런스 조정 시 자주 보는 숫자만 모은 빠른 참조용.

## Dispatch 효율 (방치형)

| 시간           | 보상 효율 | 효율/초 |
| -------------- | --------- | ------- |
| 60s (1분)      | 100%      | 1.000   |
| 1,800s (30분)  | 88%       | 0.880   |
| 7,200s (2시간) | 75%       | 0.750   |

→ 짧게 반복할수록 보상 효율 우대. 활성 플레이 시 1시간 환산 3,600 vs 2시간 1회 방치 2,700 (≒ 1.33x).
→ 전투 로그는 7,200턴까지 풀 보존 (구버전 28,800턴 세이브에서만 트림).

### 보물 (`region.treasure`) — 분당 1회 독립 굴림 (`docs/22`)

`chance` 의미: **60초당 굴림 확률** (기존 dispatch당 1회 → 분당 1회로 재정의, 값 자체는 보존).

```
rolls = max(1, floor(durationSec / 60))
hits  = Σ(roll < region.treasure.chance) for roll in rolls
gold  = treasure.gold × hits × finalMult   (finalMult가 효율 곡선 자동 적용)
```

| 길이   | 굴림수 | 평원(5%) 기대 적중 | 페이아웃 mult | 1초당 보물 골드 |
| ------ | -----: | -----------------: | ------------: | --------------: |
| 60s    |      1 |               0.05 |          1.00 |          0.0833 |
| 1,800s |     30 |               1.50 |          0.88 |          0.0733 |
| 7,200s |    120 |               6.00 |          0.75 |          0.0625 |

→ 1초당 보물 골드 비율 = per-kill 효율 비율 = **1.00 : 0.88 : 0.75**. 검증 PASS: `scripts/treasure-rate-sim.ts` (RUNS=5000, 오차 ±2%).

### 탐험 조기 종료 (`docs/18`)

`elapsedSec / playbackSec` 비례 부분 보상. 보스/5초 미만은 보상 0. 출발 시점 길이의 효율을 그대로 유지(7200초@1800초 cancel = fresh 1800초의 0.852배). 검증 PASS: `scripts/cancel-partial-sim.ts`.

## 데미지 공식 요약

```ts
// 물리
boostedAtk = ATK × (1 + atkBoostPct)
effectiveDef = enemyDef × (1 − totalPierce)
dmg = max(1, boostedAtk − effectiveDef)
dmg ×= (0.9 + Math.random() × 0.2)         // ±10% 분산
if 크리: dmg ×= critMult                    // 도적 ×2 / 어쌔신 ×2.5
if multiplier: dmg ×= multiplier            // 스킬 멀티
if damageAmpPct: dmg ×= (1 + amp)
floor

// 마법 (skill 발동만)
baseHit = max(1, INT × intMult − MDEF)      // ignoreMdef면 MDEF=0
fireballDmg += baseHit × hits
finalMagic = floor(fireballDmg × (1 + amp) × (1 + magicAmp))

// DOT (매 턴 tick)
dotDmg = max(1, INT × 0.5 × stacks × (1 + dotAmpPct))
// DEF/MDEF 무시. 매 턴 stacks −1.

// 적 → 플레이어
dmg = max(1, eAtk − pDef)
dmg ×= (0.9 + Math.random() × 0.2)
if damage_reduction passive: dmg ×= (1 − pct)

// 회피
dodge = min(0.6, AGI × 0.002 + flat)
```

## 시뮬레이션 시 자주 체크하는 시나리오

1. **DPS 측정**: 훈련장에서 30턴 누적 데미지 / 30 → DPS/턴
2. **빌드 비교**: 같은 레벨에서 2차 직업별 훈련장 누적
3. **보스 처치 가능 시간**: 보스 HP / 예상 DPS = 턴 수 (≤30이어야 가능)
4. **회피 빌드 검증**: AGI 90+ → 18% 회피 + AGI 보정 크리
5. **DOT 빌드**: 맹독술사 + 독칼 + 도트 누적 시 INT×0.5×7 (캡+2) × dot_amp 합
