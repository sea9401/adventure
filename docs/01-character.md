# 캐릭터 시스템

> 코드: `src/lib/game/data.ts` (CLASSES, ADVANCED_CLASSES, expForLevel), `src/lib/game/logic.ts` (computeStats, applyExp)

## 직업 (1차)

레벨 1부터 선택 가능. 각 직업은 베이스 스탯, 성장 스탯, 패시브를 가집니다.

| 직업   | 컨셉           | 베이스 (HP/ATK/DEF/MDEF/SPD/AGI/INT) | 주속성 베이스 (STR/VIT/MATK) | 패시브            |
| ------ | -------------- | ------------------------------------ | ---------------------------- | ----------------- |
| 전사   | 정면 탱커/딜러 | 100/10/8/2/7/2/2                     | 5/5/0                        | 받는 데미지 −15%  |
| 도적   | 회피·치명타    | 60/16/4/2/5/18/2                     | 3/2/0                        | 크리티컬 15% (×2) |
| 마법사 | 방어 무시 마법 | 50/12/3/5/8/3/14                     | 1/1/5                        | 적 DEF 50% 무시   |

레벨업 시 성장 스탯이 누적됩니다. 직업 전환 가능(상시).

## 직업 (2차) — Lv 100 + 전직 후

각 1차 직업당 2개의 분화. 2차 직업은 1차의 성장에 `growMult` 배율과 추가 보너스(`growBonus`)를 적용합니다.

| 2차 직업 | 부모   | 성장 멀티                    | 패시브                                    |
| -------- | ------ | ---------------------------- | ----------------------------------------- |
| 광전사   | 전사   | atk×1.05, hp×0.6             | ATK +45% 상시, 매 턴 maxHP 2% 자기 데미지 |
| 방패병   | 전사   | hp×1.05, def×1.4, atk×0.55   | 받는 데미지 −15%, DEF×40% 반사            |
| 어쌔신   | 도적   | agi×0.95, atk×0.85           | 크리티컬 50% (×2.5)                       |
| 맹독술사 | 도적   | agi×0.85, growBonus int×1.05 | 도트 데미지 +50%, 독 스택 캡 +2           |
| 원소술사 | 마법사 | int×1.1, mdef×0.9            | 마법 데미지 +60%, 매 턴 INT×0.3 마력 분출 |

2차 직업 부모 매핑은 `ADVANCED_CLASS_PARENTS`에 정의되어 있습니다.

## 스탯 체계

### 주속성 → 2차 속성 환산

`computeStats`에서 다음과 같이 폴드됩니다:

- **STR** → ATK에 +1
- **VIT** → DEF에 +1
- **MATK** → INT에 +1

UI에서는 별도 표시되지만 데미지 공식상 사실상 ATK/DEF/INT와 동일하게 작동합니다.

### 2차 속성

| 스탯 | 설명                                                                     |
| ---- | ------------------------------------------------------------------------ |
| HP   | 최대 체력                                                                |
| ATK  | 물리 공격력                                                              |
| DEF  | 물리 방어력                                                              |
| MDEF | 마법 방어력                                                              |
| SPD  | 행동 슬롯 / 추가 공격 횟수 (`SPD × 0.05`만큼 추가 공격 풀)               |
| AGI  | 회피율 (`AGI × 0.002`, 최대 60%) + 크리율 보정 (`AGI × 0.001`, 최대 30%) |
| INT  | 마법 / 도트 베이스 (DOT는 `INT × 0.5 × stacks`)                          |

### 스탯 계산식

`computeStats(character)` 결과:

```
final = (cls.base + (cls.grow × m + bonus) × lvBonus) + 장비 + 세트 + 코덱스 + 명예의 전당
```

- `m`: 2차 직업의 `growMult` (없으면 1)
- `bonus`: 2차 직업의 `growBonus` (없으면 0)
- `lvBonus`: 레벨에 비례한 누적 성장 배수
- 명예의 전당 보너스는 `getMonumentBonus`로 계산되어 누적

## 레벨 / 경험치

| 항목                    | 값                                                  |
| ----------------------- | --------------------------------------------------- |
| 1차 만렙                | 100                                                 |
| 2차 만렙                | 200                                                 |
| Lv 1 ~ 99 EXP 요구량    | `level × 50`                                        |
| Lv 100 ~ 199 EXP 요구량 | `level × 50 × (1 + (level − 100) / 15)` (점진 가속) |
| Lv 100→200 누적         | 약 3.5M EXP (≈ 직전까지의 4.7배)                    |

마일스톤 토스트가 50/100/130/160/200에서 표시됩니다.
