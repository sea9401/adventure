# 2차 전직 레벨 캡 150 하향 개편안

> 관련 코드: `src/lib/game/data.ts` (`ADVANCED_LEVEL_CAP`, `level_progress` 업적, `expForLevel`), `src/lib/game/logic.ts` (`LEVEL_MILESTONES`, `getLevelCap`), `src/app/page.tsx` (도움말/마일스톤 모달 텍스트)
>
> 상태: **적용 완료** (커밋 `2d79ffd` v10). 본 문서는 변경 이력/근거 기록용.

## 1. 현재 시스템 요약

| 항목                            | 현재 값                                               | 위치                         |
| ------------------------------- | ----------------------------------------------------- | ---------------------------- |
| 1차 직업 레벨 캡                | 100                                                   | `data.ts:LEVEL_CAP`          |
| 2차 전직 레벨 캡                | **200**                                               | `data.ts:ADVANCED_LEVEL_CAP` |
| 2차 전직 가능 시점              | 100                                                   | (전직 퀘스트)                |
| 레벨 마일스톤 모달              | 50 / 100 / 130 / 160 / 200                            | `logic.ts:LEVEL_MILESTONES`  |
| `level_progress` 업적 최종 단계 | 200 (심연 정복자)                                     | `data.ts:level_progress`     |
| EXP 곡선                        | `level < 100`: `L*50` / `≥100`: `L*50*(1+(L-100)/15)` | `data.ts:expForLevel`        |

**100 → 200 누적 EXP**: 약 3.5M (직전 ~4.7배). 후반 컨텐츠 페이스를 길게 가져가려는 의도가 주석에 명시되어 있음.

## 2. 변경 목표

`ADVANCED_LEVEL_CAP` 200 → **150** 으로 하향.

- 100~200 구간 그라인드가 후반 페이스를 늘어지게 한다는 판단.
- `docs/16-level-cap-150-plan.md` 사전 검증(별도 대화)에서 lv 150 풀빌드라도 일반 사냥/필드 보스 거의 무리 없음을 확인. 심연 군주 솔로 raw clear는 lv 200에서도 어려운 영역이라 50렙 차이가 본질적 게이팅을 만들지 않음.
- 캡 하향에 맞춰 **마일스톤 / 업적 / 도움말** 등 200을 가정한 컴포넌트를 함께 정리.

## 3. 코드 변경 항목

### 3.1 상수 (`src/lib/game/data.ts`)

```ts
- export const ADVANCED_LEVEL_CAP = 200;
+ export const ADVANCED_LEVEL_CAP = 150;
```

`LEVEL_CAP = 100` (1차 캡)은 그대로 유지.

### 3.2 EXP 곡선 (`expForLevel`)

**현재 의도**: 100~200 구간을 천천히 늘려 후반 페이스 보존.

**옵션 A (단순 캡 변경 — 권장)**

- 곡선은 그대로 두고 캡만 150으로 자른다.
- 100 → 150 구간 누적 EXP ≈ 다음 표 참조.
  - `expForLevel(100) = 5000`
  - `expForLevel(150) = 150*50*(1+50/15) = 32,500`
  - 합산 100→150 누적 ≈ 약 880k EXP (기존 100→200의 ~25%)

**옵션 B (체감 페이스 보정)**

- 100~150 구간 EXP 곡선을 살짝 조정해 기존 100~200의 50~70% 정도 그라인드 시간으로 보존.
- 예: `(L-100)/15` → `(L-100)/8` 로 가중치 강화.

> 권장: **옵션 A**. 후반 페이스를 짧게 압축하는 것 자체가 변경의 의도. 추가 보정 없이 자연스럽게 끊김.

### 3.3 레벨 마일스톤 (`src/lib/game/logic.ts`)

```ts
- export const LEVEL_MILESTONES = [50, 100, 130, 160, 200] as const;
+ export const LEVEL_MILESTONES = [50, 100, 125, 150] as const;
```

- 130/160/200 → 125/150으로 압축.
- 100을 2차 전직 unlock 마일스톤, 125는 2차 진행 절반 지점, 150은 최종 마일스톤.

### 3.4 `level_progress` 업적 재설계 (`src/lib/game/data.ts`)

기존 10단계 → **8단계**로 축소.

| 단계        | 기존 목표 | 기존 보상              | **변경 목표** | **변경 보상**                                 |
| ----------- | --------- | ---------------------- | ------------- | --------------------------------------------- |
| 모험가      | 10        | 골드 500               | 10            | 골드 500                                      |
| 숙련자      | 20        | 1,500                  | 20            | 1,500                                         |
| 정예        | 35        | 4,000                  | 35            | 4,000                                         |
| 전사        | 50        | 10,000                 | 50            | 10,000                                        |
| 베테랑      | 70        | 25,000                 | 70            | 25,000                                        |
| 영웅        | 90        | 60,000                 | 90            | 60,000                                        |
| 초월자      | 100       | 120,000                | 100           | 120,000                                       |
| 각성자      | 130       | 250k + 그림자 조각 ×3  | **125**       | **250k + 그림자 조각 ×3**                     |
| 심연 도전자 | 160       | 500k + 공허의 정수 ×3  | — (제거)      | —                                             |
| 심연 정복자 | 200       | 1M + 심연 군주의 핵 ×1 | **150**       | **600k + 공허의 정수 ×3 + 심연 군주의 핵 ×1** |

- 7단계까지는 그대로 유지 (1차 진행은 영향 없음).
- 8단계(각성자) 130 → **125** 로 당김. 마일스톤(125)과 정렬.
- 9단계 제거.
- 10단계(심연 정복자) 200 → **150** 으로 당김. 보상은 기존 9·10단계를 묶어 정복자에 합산 (정수 + 군주의 핵). 골드 1M → 600k 로 약간 압축.

`AchievementId` 유니언은 변경 없음 (단계 개수만 감소). 기존 `claimedTiers["level_progress"]` 값이 8/9/10 인 세이브가 있으면 자동으로 8까지로 클램프되므로 마이그레이션 보호 필요 (§5 참고).

### 3.5 도움말 / 안내 텍스트 (`src/app/page.tsx`)

- "200렙", "심연 도전자/정복자" 라벨이 들어간 도움말 섹션이 있다면 함께 갱신.
- 헬프 모달의 레벨 캡 / 2차 전직 설명을 `최대 lv 150` 으로 수정.
- 통계 패널에서 `highestLevel` 표시 자체는 변경 없음.

> 검색 키워드: `200렙`, `200 도달`, `심연 정복자`, `LEVEL_MILESTONES`.

### 3.6 명예의 전당 (`Monument`) — 변경 없음

`MONUMENT_LEVEL_FACTOR(lv) = lv * 0.05`, `MONUMENT_MAX_LEVEL = 25` 는 캐릭터 레벨과 무관 (마을 건물 레벨). 영향 없음.

## 4. 변경 안 하는 것 (의도)

| 항목                           | 이유                                                                                                               |
| ------------------------------ | ------------------------------------------------------------------------------------------------------------------ |
| `LEVEL_CAP = 100`              | 1차 직업 캡은 2차 전직 게이팅으로 그대로 의미가 있음.                                                              |
| 보스 스탯                      | lv 150 풀빌드로 균열의 수호자까지 클리어 가능. 심연 군주는 어차피 풀세팅·스킬 의존이라 캡 하향으로 본질 변화 없음. |
| 스킬 unlockLevel               | 1차 후반 마지막이 100, 2차는 모두 100 unlock. 150 캡 안에서도 모든 스킬 사용 가능.                                 |
| 도감 / 명예의 전당 보상 곡선   | 레벨과 직접 연동되지 않음.                                                                                         |
| 장비 세트 / 보스 유니크 / 반지 | 레벨 무관.                                                                                                         |
| 협동 보스 / 길드 / 영지        | 영향 없음.                                                                                                         |

## 5. 마이그레이션 / 호환성

**방침: 마이그레이션 로직을 작성하지 않는다.**

이 변경을 다음 버전 컷의 시작점으로 잡고, 세이브를 한 번 초기화한 뒤 새 캡(150)으로 출발한다.

- `character.level > 150`, `claimedTiers["level_progress"] > 8` 같은 잔재를 다룰 변환 코드를 추가하지 않음 → `data.ts` 상수만 갈아엎으면 끝.
- 기존 캐릭터/세이브가 있는 사용자에게는 **인-게임 공지 또는 변경 사항 노트로 초기화 안내**가 필요. 어떤 채널로 알릴지는 §6 참고.
- 초기화 자체는 zustand persist key 변경 (또는 store version bump) 으로 처리. 기존 키는 자연 폐기.
