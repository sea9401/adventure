# 콘텐츠 로드맵

게임에 추가할 콘텐츠 후보 목록. 우선순위는 변동 가능 — "1·2·3 부터" 표기는 한 번에 묶어 진행하자고 합의한 1차 배치.

## 🎯 미회수 떡밥 — 1차 배치

### 1. 카이 떡밥 — "호수에 뭔가 있다"
- 안개 호수 가장 깊은 곳에 새 보스/이벤트.
- 카이 후속 의뢰 — 수지 퀘스트 완료 후 해금.
- 톤: 따뜻한 남편 톤은 유지하되, "수지에게는 비밀이지만…" 식으로 자연스럽게 분기.
- storyFlag 로 진행 추적.

### 2. 후드 쓴 손님 (`diola_stranger`)
- 폐허(ruins) 잠금 해제 트리거.
- 카이 떡밥 들은 상태에서 → "옛 폐허에 안내해 줄까?" 제안 → 수락 시 해금.
- 기존 greeting `"호수 너머를 봤을 때, 다시 와"` 가 이미 트리거 조건을 암시.

### 3. 잠긴 옛 폐허 (`ruins`)
- 가장 크게 비어 있는 신규 지역 슬롯.
- 새 몬스터 2-3종 (`undead` / `golem` 태그 활용 — 정의돼 있는데 미사용).
- 새 재료 1-2종 + 새 장비 1-2종.
- 적정 레벨 8-10 가정.

---

## 🏘️ 디올라 마을 살 붙이기 — 2차 후보
디올라엔 NPC 만 있고 의뢰가 0건. 종착점 마을이 휑함.

### 4. 디올라 길드 의뢰 라인
- 노라(여관): "묵으러 온 손님이 짐을 두고 사라졌다" → 호수/폐허 탐색 훅.
- 보로(잡화상): "거미줄이 자꾸 부족하다" 같은 반복형 채집 의뢰.
- 리오(꼬마): 짧은 심부름 → 칭호("동네 형/누나").

### 5. 디올라 훈련/대장간
- 디올라 전용 교관 또는 호수 테마의 새 스킬/제작법.
- 마을별 복귀 지점 시스템과 시너지 — 디올라를 거점으로 쓰는 이유가 생김.

---

## ⚒️ 최근 시스템 확장 — 3차 후보

### 6. 제작서 드롭 추가 배포
- 슬라임 코어 1% 같은 기존 패턴에 곁들여, 보스급/희귀몹에 새 제작서 1-2개씩.
- 후보: 들개 → 가벼운 가죽 부츠(spd), 거미 → 끈끈이 망토(evade), 박쥐 → 야안 부적(crit).

### 7. 장비 강화 라인업 (못박힌 야구방망이 시리즈)
- "기존 장비 + 재료 → 강화" 패턴 확장.
- 후보: 가죽갑옷 + 슬라임핵 → 점착 가죽갑옷, 활력의 반지 + 요정가루 → 풍요의 반지.

---

## 🗺️ 새 지역/몬스터 — 추후

### 8. 호수 깊은 곳 (sub-region)
- lake 의 시련 끝에 별개 노드. 카이 떡밥 보스.

### 9. 새 태그 활용 — `undead` / `golem` / `dragon`
- 타입 정의는 있는데 사용 안 됨. ruins 가 자연스러운 도입처(언데드/골렘) — dragon 은 더 후반.

---

## 🎮 시스템/UX 마이너 개선

### 10. 칭호 추가
- "방랑자"(전 지역 방문), "도감 마스터"(전 몬스터 처치), "고향 떠난 자"(디올라를 복귀 지점으로).

### 11. 자동 사냥 스탯 — 처치/획득 누적 표시
- 도감과 별개로, 현재 세션의 짧은 누적 패널.
